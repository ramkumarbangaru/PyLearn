1) Unzip the VSCode Portable file and place in C:\ (or anywhere else you like)
2) Run VSCodePortable.exe (We are good with VS code here)
3) We need python extension to work with python better. to install python extension follow below steps
3.1) Click on Extension icon, 5th(last icon on left side from top  to bottom)
3.2) Select 3 dots (...) (more) options on extension window(not the one on very right most)
3.3) Select install from vsix from the menu
3.4) select the python.vsix file 
3.5) once installation completed message appeared, close and reopen the editor.

We are all done with VS Code here. Any doubts please feel free to reach me.
