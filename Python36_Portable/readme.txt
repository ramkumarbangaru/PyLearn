1) download python36.zip and extract at C:\ drive
2) Open Environment variables using either one of below steps

2.1)  Right click 'My Computer' and select 'Properties'.
2.2)  Click 'Advanced System Settings' link.
2.3) Click 'Advanced' tab.
2.4) Click 'Environment Variables...' button.

3) Find path variable and click edit 
3.1) add below paths at the end, make sure you have given semi colon to split them
;C:\Python36\;C:\Pytho36\Scripts;

4) Apply and close
5) Close all opened cmd and Open Cmd again. now type python.
